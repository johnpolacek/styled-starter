Styled Starter is built on top of Styled System and Next.js, using Catalog for the design system styleguide.

Styled Components / Styled System

Styled Components is a library for writing CSS directly in components. Styled System is a utility that hooks style props from a design theme into components. Use them together to inject components with a design system.

As you build components, keep them documented in the styleguide, powered by Catalog.

Next.js

Next.js is a minimalistic framework for server-rendered React applications. Using Next.js allows us to have the developer experience of a Single Page App while delivering the performance gains of Server Side Rendering.
